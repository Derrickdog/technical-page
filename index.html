Introduction

HTML stands for Hyper Text Markup Language. It is the standard markup language for creating Web pages. HTML describes the structure of a Web page. It consists of series of elements. Elements tell the browser how to display the content. They also label pieces of content.

HTML documents follow a specific outline to be displayed on a Web browser.

<!DOCTYPE html> <html lang="en"> <head>   <title>Title</title> </head> <body>   <h1>Hello World</h1> </body> </html>
Headings

HTML headings are titles or subtitles that you want to display on a webpage.

<h1>Heading 1</h1> <h2>Heading 2</h2> <h3>Heading 3</h3> <h4>Heading 4</h4> <h5>Heading 5</h5> <h6>Heading 6</h6>

Paragraphs

Paragraphs in HTML always start on a new line and are usually blocks of text.

<p>This is an example of a paragraph</p>

This is an example of a paragraph

Links

Links allow users to travel to external or internal webpages by clicking the link.

Links use the "a" tag, which stands for anchor, and the "href" attribute, which stands for hypertext reference.

<a href="https://www.google.com">Google</a> Google
Images

Images display pictures of various file formats to the webpage

Image tags are self closing and use the "src" attribute to specify the path of the image file. The "alt" attribute provides alternative text that describes the image if it cannot be displayed.

<img src="photo.png" alt="Crystal ball">
Lists

Unordered lists use the "ul" tag and "li" tag for list items. List items are nested in the unordered list tag.

<ul>   <li>Item 1</li>   <li>Item 2</li>   <li>Item 3</li> </ul>
  • Item 1
  • Item 2
  • Item 3

Ordered lists use the "ol" tag and "li" tag for list items. List items are nested in the ordered list tag.

<ol>   <li>Item 1</li>   <li>Item 2</li>   <li>Item 3</li> </ol>
  1. Item 1
  2. Item 2
  3. Item 3
