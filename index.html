Introduction

HTML stands for Hyper Text Markup Language. It is the standard markup language for creating Web pages. HTML describes the structure of a Web page. It consists of series of elements. Elements tell the browser how to display the content. They also label pieces of content.

HTML documents follow a specific outline to be displayed on a Web browser.

<!DOCTYPE html> <html lang="en"> <head>   <title>Title</title> </head> <body>   <h1>Hello World</h1> </body> </html>
